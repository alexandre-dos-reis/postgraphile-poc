# PostGraphile POC

This Project is a POC showing the `Postgraphile` capabilities. it's a NodeJS package that allows instant GraphQL API server from a PostGres Database.

A tutorial can be follow [here.](https://github.com/graphile/postgraphile/blob/main/examples/forum/TUTORIAL.md#installing-postgres)

# Start

- Run the dev server with `pnpm dev`
- The Postgraphile Graphql endpoint is available at `http://localhost:7000/graphql`
- The GraphiQL endoint is available at `http://localhost:7000/graphiql`
- The Postgraphile options enable graphql hot reload upon schema changes !
